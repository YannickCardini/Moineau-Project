# Projet Moineau

## Requierement:

* nodejs
* mysql (mariadb 10.2 or above)


## Installation

* git clone
* execute: `bdd/setup.sql ` in mysql command line:`mysql> source bdd/setup.sql;`
* execute: ` npm install`
* run the app with `node bin/www`

##Description

Upload, view and edit your XML file.
